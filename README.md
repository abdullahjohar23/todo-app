# 📝 ToDo App (Flutter)

A simple and clean ToDo application built using Flutter. This app allows users to add, complete, and delete tasks with local data persistence using Hive.

---

## 🚀 Features

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Local storage using Hive (data persists after app restart)  
- Clean and minimal UI  
- Works on real devices and emulators  

---

## 🛠️ Tech Stack

- Flutter  
- Dart  
- Hive (Local Database)
